# List of Different Feature Extraction Tools that could be used.  

- https://github.com/gozsari/ProtFeat => extracts 31 distinct features from fasta peptide input using iFeature as part of it; works only with Python 3.9
- https://github.com/Superzchen/iFeature/blob/master/README.md => iFeature pure

    NOTE: For iFeature, I recommend using git bash.  It works fairly well and you can do basic I/O operations to get extracted feature groups from it.  
- https://bionewsdigest.medium.com/machine-learning-basics-in-omics-analysis-f67f77729671 => useful for iFeature information. 

- https://github.com/RafsanjaniHub/PyFeat/blob/master/RESULTS.md => pyFeat based feature extraction
